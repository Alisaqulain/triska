# Quick Setup Guide

## Before Running

1. **Add Your Logo**
   - Place your logo file at: `/public/logo.png`
   - Recommended: Square format, minimum 200x200px
   - Supported formats: PNG, JPG, SVG

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

## Project Features

✅ Home page with hero section, categories, and featured products
✅ Product listing page with advanced filters
✅ Responsive design (mobile, tablet, desktop)
✅ Smooth animations with Framer Motion
✅ Premium color scheme (cream, beige, gold)
✅ 3D hover effects on product cards
✅ Filter sidebar with expand/collapse animations

## Customization

- **Products**: Edit product arrays in `app/page.js` and `app/products/page.js`
- **Colors**: Modify `tailwind.config.js`
- **Content**: Update text in component files

## Notes

- All product images are currently using Unsplash placeholders
- Replace with your actual product images
- Logo is expected at `/public/logo.png`
